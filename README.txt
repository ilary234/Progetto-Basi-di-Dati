Il progetto contiene due classi main, una per l'interfaccia utente e l'altra per l'impiegato.
Per poterle eseguire sono possibili due opzioni:
1. Fare l'accesso manuale attraverso le classi AppAmministratore in model/amministratore e AppUtente in model/utente.
2. Eseguire i file jar.
